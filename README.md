# 100-css-projects
There are 100 CSS Projects in this repository, and these projects contain the CSS Properties that you will use 90% of the time in your lifetime.

![Repo Image](https://assets-global.website-files.com/5f9072399b2640f14d6a2bf4/652f1b86cbb36054876e47b0_Halloween%20Banner-p-800.jpg)

# Projects Type <br>
1-8    = Basic CSS Practice 🎨 <br>
9-49   = Components like card, navigation menu 🧩 <br>
50-60  = Animation Projects 🔮 <br>
61-70  = Flexbox & Grid projects 📦 <br>
71-89  = Landing Pages 🧾 <br>
90-100 = Big Clones like spotify, AirBnB 🚀 <br>

# Want to Contribute? <br>
1. Everyone is invited to contribute to this repository. Just remember, this repository is tailored for beginners, so refrain from using overly complex methods to shorten the code. If you have an alternative approach that is readable and beginner-friendly for any project, please feel free to contribute.<br>

2. If you encounter a project that isn't functioning properly, you can contribute by debugging it or by notifying us of the issue. 🛠️ <br>
